# Extension Icons

Place your extension icons in this folder:

- icon16.png (16x16px)
- icon48.png (48x48px)
- icon128.png (128x128px)

You can use any image editor to create these, or use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

For now, you can use a simple shield emoji as a placeholder or create proper icons.

Temporary solution: Create a colored square with a shield symbol.
